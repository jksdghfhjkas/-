# Нейросеть для классификаций состояний поезда по данным давления

LSTM нейросеть классифицирует состояни поезда по данным давления из тормозной магистрали.

**Состояния**
- 0 - движение
- 1 - торможение
- 2 - перекрыша
- 3 - отпуск
- 4 - стабилизация

# Архитектура
На вход поступаетют батчи по 5 элементов, которые хранят данные давления, первой производной, второй производной.

- Первый LSTM слой, 64 нейрона
- Второй LSTM слой, 32 нейрона
- Третий слой, 16 нейронов с функцией активаций relu
- дропаут с выключением 30% нейронов
- выход 5 нейронов с активацией softmax

Используется оптимизация Adam с learning_rate=0.00001

Функцию потерь взял - категориальную кросс-энтропию
